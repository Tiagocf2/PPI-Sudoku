Controles:

  • ou para selecionar um quadrado.

  • para apagar o quadrado selecionado.

Como Jogar:

Dado o tabuleiro inicial, você deve conseguir preencher todos os quadrados com números de 1 a 9, de modo que:

  1. Não haja números repetidos nas linha.
  2. Não haja números repetidos nas colunas.
  3. Não haja números repetidos nos quadrados maiores.

Boa sorte!
